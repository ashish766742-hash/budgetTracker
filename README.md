# Budget Tracker

## Project Description
The Budget Tracker is a user-friendly application designed to help individuals manage their finances effectively. It allows users to track income and expenses, categorize spending, and visualize their budget through comprehensive reports and graphs.

## Key Features
- **Income Tracking**: Easily input and manage different sources of income.
- **Expense Tracking**: Record and categorize all expenses to understand spending habits.
- **Budgeting Tools**: Set budget limits for various categories and monitor progress.
- **Reports and Analytics**: Generate graphs and reports to visualize financial habits and make informed decisions.
- **User Authentication**: Secure user login to protect financial data.
- **Mobile Responsive Design**: Access the application seamlessly on any device.

## Technologies Used
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express.js
- **Database**: MongoDB

## Installation Instructions
To run the application locally:
1. Clone the repository: `git clone https://github.com/ashish766742-hash/budgetTracker.git`
2. Navigate to the project directory: `cd budgetTracker`
3. Install dependencies: `npm install`
4. Start the server: `npm start`

## Usage
After starting the server, go to `http://localhost:3000` in your web browser to access the application. Create an account or log in to start tracking your budget.
